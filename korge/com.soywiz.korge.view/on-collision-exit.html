Adds a component to this that checks each frame for collisions against descendant views of root that matches the filter, when a collision is found, it remembers it and the next time the collision does not occur, callback is executed with this view as receiver, and the collision target as first parameter. if no root is provided, it computes the root of the view this each frame, you can specify a collision kind
