This is the configuration for an independent parallax layer. Independent means that these layers are not attached to the parallax plane. Their speed in X and Y direction can be configured independently by speedX and speedY. Also, their self-Speed selfSpeedX and selfSpeedY can be configured independently.

name has to be set to the name of the layer in the used Aseprite file. The image on this layer will be taken for the layer object. repeatX and repeatY describes if the image of the layer object should be repeated in X and Y direction. speedX and speedY are the factors for scrolling the parallax layer in X and Y direction relative to the game play field. selfSpeedX and selfSpeedY are the factors for scrolling the parallax layer in X and Y direction continuously and independently of the player input.
