TexturedVertexArray

class TexturedVertexArray(vcount: Int, indices: IntArray, isize: Int)
Content copied to clipboard

Allows to build a set of textured and colored vertices. Where vcount is the number of vertices and isize, the maximum number of indices.

vcount and isize could be decreased later, but not increased since the buffer is created at the beginning.
