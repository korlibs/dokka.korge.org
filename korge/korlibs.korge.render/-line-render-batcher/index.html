LineRenderBatcher

class LineRenderBatcher(val ctx: RenderContext)

A context that allows to draw lines using AG (Accelerated Graphics).

You should use it by calling:

ctx.draw(matrix) { ctx.line(0, 0, 100, 100) ctx.line(100, 100, 0, 100) // ... }
