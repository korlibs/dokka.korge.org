Historiogram

class Historiogram(out: IntIntMap = IntIntMap())

Supports getting a map determining the number of occurrences for different Int

For example:

intArrayOf(1, 1, 5, 1, 9, 5) // 3x 1, 2x 5, 1x 9
==
intIntMapOf((1 to 3), (5 to 2), (9 to 1))
