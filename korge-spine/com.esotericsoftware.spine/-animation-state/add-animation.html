Adds an animation to be played after the current or last queued animation for a track. If the track is empty, it is equivalent to calling .setAnimation.

Return

A track entry to allow further customization of animation playback. References to the track entry must not be kept after the AnimationStateListener.dispose event occurs.

Parameters

delay

If 0, sets TrackEntry.getDelay. If <= 0, the delay set is the duration of the previous track entry minus any mix duration (from the AnimationStateData) plus the specified delay (ie the mix ends at (delay = 0) or before (delay< 0) the previous track entry duration). If the previous entry is looping, its next loop completion is used instead of its duration.
