An IntervalSystem of a world with a context to entities.

It must have at least one of allOfComponents, anyOfComponent or noneOfComponents objects defined. These objects define a Family of entities for which the IteratingSystem will get active. The IteratingSystem will use those components which are part of the family config for any specific processing within this system.

Parameters

allOfComponents

is specifying the family to which this system belongs.

noneOfComponents

is specifying the family to which this system belongs.

anyOfComponents

is specifying the family to which this system belongs.

comparator

an optional EntityComparator that is used to sort entities. Default value is an empty comparator which means no sorting.

sortingType

the type of sorting for entities when using a comparator.

interval

the interval in which the system gets updated. Default is EachFrame.

enabled

defines if the system gets updated when the world gets updated. Default is true.
