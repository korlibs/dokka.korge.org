IntervalSystem

abstract class IntervalSystem(val interval: Interval = EachFrame, var enabled: Boolean = true)
Content copied to clipboard

A basic system of a world without a context to entities. It is mandatory to implement onTick which gets called whenever the system gets updated according to its interval.

If the system uses a Fixed interval then onAlpha can be overridden in case interpolation logic is needed.

Parameters

interval

the interval in which the system gets updated. Default is EachFrame.

enabled

defines if the system gets updated when the world gets updated. Default is true.
