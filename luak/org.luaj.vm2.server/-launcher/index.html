Interface to launch lua scripts using the LuajClassLoader.

Note: This class is experimental and subject to change in future versions.

*

This interface is purposely genericized to defer class loading so that luaj classes can come from the class loader.

*

The implementation should be acquired using LuajClassLoader.NewLauncher or LuajClassLoader.NewLauncher which ensure that the classes are loaded to give each Launcher instance a pristine set of Globals, including the shared metatables.

Since

luaj 3.0.1
