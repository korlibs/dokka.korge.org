class LuaScriptEngine : AbstractScriptEngine, ScriptEngine, Compilable
Content copied to clipboard

Implementation of the ScriptEngine interface which can compile and execute scripts using luaj.

This engine requires the types of the Bindings and ScriptContext to be compatible with the engine. For creating new client context use ScriptEngine.createContext() which will return LuajContext, and for client bindings use the default engine scoped bindings or construct a LuajBindings directly.
