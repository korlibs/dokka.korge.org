virtFrame
Link copied to clipboard
override val virtFrame: RectangleInt? = null
Content copied to clipboard

This defines a virtual frame RectangleInt which surrounds the bounds RectangleInt of the Bitmap. It is used in a trimmed texture atlas to specify the original size of a single texture. X and y of virtFrame is the offset of the virtual frame to the top left edge of the bounds rectangle. Width and height defines the size of the virtual frame.
