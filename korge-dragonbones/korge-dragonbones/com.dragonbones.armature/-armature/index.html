cacheFrameRate
Link copied to clipboard
common
var cacheFrameRate: Int
Content copied to clipboard

  • 动画缓存帧率，当设置的值大于 0 的时，将会开启动画缓存。 通过将动画数据缓存在内存中来提高运行性能，会有一定的内存开销。 帧率不宜设置的过高，通常跟动画的帧率相当且低于程序运行的帧率。 开启动画缓存后，某些功能将会失效，比如骨骼的 offset 属性等。
