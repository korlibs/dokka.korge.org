replaceSlotDisplay

fun replaceSlotDisplay(dragonBonesName: String, armatureName: String, slotName: String, displayName: String, slot: Slot, displayIndex: Int = -1): Boolean
Content copied to clipboard

  • 用特定的显示对象数据替换特定插槽当前的显示对象数据。 用 "dragonBonesName/armatureName/slotName/displayName" 指定显示对象数据。

Parameters

armatureName

  • 骨架数据名称。

displayIndex

  • 被替换的显示对象数据的索引。 （如果未设置，则替换当前的显示对象数据）

displayName

  • 显示对象数据名称。

dragonBonesName

  • DragonBonesData 实例的缓存名称。

slot

  • 插槽。

slotName

  • 插槽数据名称。
